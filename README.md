# 开发

```sh
gulp
# use example in specific path, such as PieChart/index.ts
gulp --path=PieChart
#  or gulp --path=PieChart/index.ts
```

- run ‘gulp'
- 在 `src/index.ts` 编写你的代码，浏览器会实时自动刷新，详细可以参看demo中的例子
